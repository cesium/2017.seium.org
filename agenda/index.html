BITalino… a biosignals (r)evolution

talk

DI A2
14:00-15:00
Hugo Silva
Plux

Nesta talk vamos abordar BITalino (r)evolution, um conjunto de ferramentas único composto por hardware e software de baixo preço, e criada para permitir que qualquer um crie projetos interessantes e aplicações envolventes dados fisiológicos. O hardware consiste num sistema de aquisição de sinais biológicos sem fios que pode ser usado para armazenar dados em tempo real, interface com outros dispositivos (por exemplo Arduino's ou Raspberry PI's), ou realizar portotipagem rápida de aplicativos de usuário final. O software compreende uma serie de API's programaveis, um conjunto de ferramentas de processo biológico, e uma framework para aquisição de dados e pós-processamento em tempo real.

Carlos Soares

The rise and fall of big data (and why you shouldn't care)

talk

DI A2
14:00-15:00
Carlos Soares

Depois de muito buzz e expectativas exageradas, começam a aparecer os primeiros sinais de desilusão com o Big Data. Nada disto é novo, já que a aplicação de técnicas avançadas de análise de dados nas organizações tem tido uma história turbulenta de entusiasmo e declínio. Tivemos o data mining nos anos 90, depois business intelligence, analytics e big data, que parece agora estar a ser substituído pelo renascido machine learning. Esta nova tendência está em grande parte relacionada com o desenvolvimento das novas abordagens de redes neuronais, conhecidas como deep learning. No entanto, mais buzz, menos hype, parece ser indiscutível que esta área, cujo nome mais preciso talvez deva ser "data science and engineering”, está cá para ficar. Nesta apresentação vamos tentar dar uma visão tão objetiva quanto possível da área, dos problemas que podem ser tratados, dos dados que podem ser usados, dos desafios e dos fatores que favorecem ou prejudicam o sucesso dos projetos.

Roberto Cortez

Java EE 7 meets Java 8

workshop

DI A2
14:00-15:30
Roberto Cortez

Os últimos anos tem sido bastante excitantes para Java. Novas versões de Java EE e plataformas Java SE foram introduzias em 2013 e 2014. Para Java EE, temos agora suporte de primeira class para padrões web, incluindo uma nova API para construir WebSockets, uma nova API para analisar, processar e criar JSON e um novo Cliente API em JAX-RS 2 para invocar serviços RESTful. O muito esperado processador API Batch esta agora disponível para construir aplicações de processos batch usando todas as capacidades da propria plataforma. Por outro lado, Java SE trouxe nos as muito esperadas expressões lambda para simplificar programação funcional; a nova e poderosa Streams API permite executar operações numa sequencia de elementos, como filtragem, mapeamento, ordenação e muitas outras; uma novíssima Date Time API, torna mais fácil para lidar como fusos horarios e periodos de tempo. Vamos combinar todos os elementos e ver como é divertido desemvolver aplicações em Java.
